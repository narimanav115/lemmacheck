Напиши Python desktop-приложение для полнотекстового поиска по леммам с графическим интерфейсом.

## Технологии:
- UI: tkinter (ttk для современных виджетов)
- Русская лемматизация: pymorphy3
- Английская лемматизация: nltk + WordNetLemmatizer
- Чтение DOCX: python-docx
- Чтение PDF: PyMuPDF (fitz)
- Чтение TXT: встроенный open()

## Функционал приложения:

### 1. Главное окно:
- Кнопка "Добавить файлы" - выбор DOCX, PDF, TXT (множественный выбор)
- Кнопка "Добавить папку" - рекурсивный обход всех документов в папке
- Список загруженных документов с возможностью удаления
- Поле ввода поискового запроса
- Кнопка "Искать"
- Область результатов с подсветкой найденных слов
- Статусбар с количеством документов и индексированных слов

### 2. Движок поиска (класс LemmaSearchEngine):
- Автоопределение языка слова (кириллица → pymorphy3, латиница → nltk)
- Метод add_document(doc_id, text, filename) - индексация документа
- Метод search(query) - поиск с TF-IDF ранжированием
- Метод lemmatize(text) - лемматизация текста
- Хранение: inverted index (лемма → список документов и позиций)

### 3. Парсеры документов:
- extract_text_from_docx(filepath) - извлечение текста из DOCX
- extract_text_from_pdf(filepath) - извлечение текста из PDF  
- extract_text_from_txt(filepath) - чтение TXT с автоопределением кодировки

### 4. Отображение результатов:
- Показывать: имя файла, релевантность (score), фрагмент текста с контекстом
- Подсветка найденных слов в результатах
- Двойной клик - открыть файл в сиaстемном приложении
- Сортировка по релевантности

### 5. Дополнительно:
- Прогрессбар при индексации больших файлов
- Многопоточность: индексация в отдельном потоке чтобы UI не зависал
- Обработка ошибок: битые файлы, неподдерживаемые кодировки
- Сохранение/загрузка индекса в JSON для быстрого старта

## Пример использования:
1. Пользователь нажимает "Добавить файлы"
2. Выбирает несколько DOCX и PDF
3. Приложение индексирует их (показывает прогресс)
4. Пользователь вводит "учебный план" или "student study"
5. Приложение находит все документы где есть эти слова в любой форме
6. Результаты отсортированы по релевантности с фрагментами текста
